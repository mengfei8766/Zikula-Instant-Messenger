Install How-To

  • Step 1: Copy files to Zikula
    • Copy the "Zim" folder into the "modules" folder on your zikula install.
  • Step 2: Install ZIM
    • Log into the Administration panel of your Zikula site. navigate to the "Extensions Manager" module. Locate the module "Zim" and click on install.
  • Step 3: Add the block
    • Log into the Administration panel of your Zikula site. navigate to the "Blocks" module.
      Next you want to add a new block, from the list of blocks select "Zikula instant messenger".
      Chose the block position of where you wish the contact list to be displayed.
  • Step 4: Use ZIM
    • Navigate to your site where the block should be displayed. Enjoy.
